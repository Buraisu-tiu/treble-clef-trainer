## Deploy to GitHub Pages
1. Go to repository Settings > Pages
2. Under "Build and deployment", select:
   - Source: "Deploy from a branch"
   - Branch: "main" / "/ (root)"
3. Click Save
4. Wait for deployment (~2-3 minutes)
